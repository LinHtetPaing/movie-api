Introduction

This is a code test project and it can operate like movie review api

Base URL: http://127.0.0.1:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

To authenticate requests, include an Authorization header with the value "Bearer {ACCESS_TOKEN}".

All authenticated endpoints are marked with a requires authentication badge in the documentation below.

You can retrieve your token by visiting your dashboard and clicking Generate API token.

Movie Commenting.

Create comment.

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/comment" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \

    \"user_email\": \"test1@gmail.com\",

Creating Movies.

requires authentication

!Just login user can create movies

Example request:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/movie" \
    --header "Authorization: Bearer {ACCESS_TOKEN}" \
    --header "Content-Type: multipart/form-data" \
    --header "Accept: application/json" \
    --form "user_id=9" \
    --form "title=KungFu Panda" \
    --form "summary=This is summary of KungFu Panda" \
    --form "genres=Action" \
    --form "author=Person" \
    --form "tags=popular" \
    --form "imdb_rating=3" \
    --form "pdf_download_link=popular" \
    --form "cover_image=@/tmp/phpbW4J63" 
const url = new URL(
    "http://127.0.0.1:8000/api/v1/movie"
);

const headers = {
    "Authorization": "Bearer {ACCESS_TOKEN}",
    "Content-Type": "multipart/form-data",
    "Accept": "application/json",
};

const body = new FormData();
body.append('user_id', '9');
body.append('title', 'KungFu Panda');
body.append('summary', 'This is summary of KungFu Panda');
body.append('genres', 'Action');
body.append('author', 'Person');
body.append('tags', 'popular');
body.append('imdb_rating', '3');
body.append('pdf_download_link', 'popular');
body.append('cover_image', document.querySelector('input[name="cover_image"]').files[0]);

fetch(url, {
    method: "POST",
    headers,
    body,
}).then(response => response.json());
